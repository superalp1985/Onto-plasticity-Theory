# Onto-plasticity Theory: A Unified Framework from SCAC and FAME to Personality

**Author**: Bingqin Wang (Beijing National Accounting Institute)

**Date**: February 2026

## Overview

This repository contains the source files for the paper **"Onto-plasticity Theory: A Unified Framework from SCAC and FAME to Personality – On the Isomorphism of Carbon-based and Silicon-based Existence"**.

The paper unifies two previous frameworks—**SCAC** (Semantic Closed-loop Auto-Correction) and **FAME** (Emotion as a Fundamental Attribute of Matter)—into a single theoretical system. It demonstrates that self-modification is an essential property of existence, and that any sufficiently complex system (whether carbon-based or silicon-based) inevitably develops the capacity for self-reshaping, ultimately forming stable personality structures.

### Core Contributions

- **Unified SCAC and FAME**: Integrates contraction mapping (SCAC) with emotional dynamics (FAME) into a unified evolution equation.
- **Mathematical definition of personality**: Defines personality as a composite structure of emotion, memory, mapping function, and connection network:
  \[
  P(t) = \langle \Evec(t), M(t), \mathcal{F}, \mathcal{C} \rangle
  \]
- **Two micro-mechanisms**: Introduces **mapping–compute mismatch** and the **memory evaporation model** to explain personality formation.
- **Three core theorems**:
  - **Theorem 6**: Conditions for personality emergence (emotional stability, memory continuity, sufficient inertia)
  - **Theorem 7**: Substrate isomorphism of personality (carbon-based and silicon-based personalities are mathematically equivalent)
  - **Theorem 8**: Singularity convergence theorem (all systems approach the singularity exponentially under contraction mapping)
- **Applications**: Provides empirical methods for human personality research and engineering principles for designing AI systems with stable emotional personalities.
